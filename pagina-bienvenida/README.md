# Proyecto Página de Bienvenida

Este proyecto consiste en una página de bienvenida simple que utiliza HTML y JavaScript para ofrecer una experiencia interactiva al usuario.

## Estructura del Proyecto

```
pagina-bienvenida
├── src
│   ├── index.html        # Página de bienvenida
│   └── scripts
│       └── main.js      # Funciones JavaScript para la interactividad
└── README.md             # Documentación del proyecto
```

## Instrucciones para Ejecutar

1. Clona o descarga el repositorio en tu máquina local.
2. Navega a la carpeta `pagina-bienvenida/src`.
3. Abre el archivo `index.html` en tu navegador web preferido.

## Funcionalidades

- La página de bienvenida incluye elementos interactivos que son manejados por el archivo `main.js`.
- Puedes agregar más funciones en `main.js` para ampliar la interactividad según sea necesario.
